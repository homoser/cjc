创客起点---Arduino

Arduino

       Arduino是什么？   Arduino是一款便捷灵活、方便上手的开源电子平台，包含硬件（各种型号的开发板）和软件（集成开发环境）两个部分，完全开源。
       Arduino做什么？   爱好者和开发者使用Arduino，快速作出具有智能互动的电子产品。例如智能机器人、智能避障循迹小车、物联网电子设计、仿真机械臂、智能家居、智慧农业。。。
       谁来学Arduino？   电子设计爱好者、发明创造爱好者、科技创新参赛者
       
       Arduino四大特点   全平台、简单化、开放性、发展性
       Arduino大家族     硬件 Arduino Uno Nano Mega2560   
                         软件 Arduino IDE   VS Code
